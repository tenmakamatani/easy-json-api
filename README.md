# json api server

api server to test frontend

## How to run

```
$ git clone https://github.com/tenmakamatani/json-api-server.git
$ cd json-api-server
$ yarn (or npm install)
$ yarn start (or npm run start)
```

## Endpoint

```
GET -> http://localhost:3000/todos
GET -> http://localhost:3000/todos/:id
```
